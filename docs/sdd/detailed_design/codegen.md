# 目标代码生成设计

## 1. 模块概述

目标代码生成器是CScriptEngine的后端组件，负责将中间代码（四元式）转换为目标平台的机器码。它是编译器的第五个阶段，为后续的JIT执行提供输入。

## 2. 设计原理

目标代码生成器通过遍历中间代码（四元式），为每个四元式生成对应的目标平台机器码。它需要考虑目标平台的指令集、寄存器分配、内存管理等因素。

## 3. 数据结构

### 3.1 目标代码

目标代码是目标代码生成器的输出，包含：
- **机器指令**：目标平台的机器指令
- **数据段**：存储全局变量和常量
- **代码段**：存储生成的机器码

### 3.2 寄存器分配

寄存器分配是目标代码生成的重要部分，包括：
- **寄存器类型**：通用寄存器、浮点寄存器等
- **寄存器分配策略**：基于图着色的寄存器分配等
- **寄存器溢出处理**：当寄存器不足时的处理策略

### 3.3 指令选择

指令选择是选择合适的机器指令来实现中间代码的过程：
- **指令集**：目标平台的指令集
- **指令格式**：指令的编码格式
- **指令优化**：选择高效的指令序列

## 4. 算法流程

1. **初始化**：准备中间代码，设置初始状态
2. **遍历中间代码**：遍历四元式序列
3. **指令选择**：为每个四元式选择合适的机器指令
4. **寄存器分配**：为变量和临时值分配寄存器
5. **内存管理**：处理全局变量和局部变量的内存分配
6. **代码生成**：生成目标平台的机器码
7. **输出**：返回生成的目标代码

## 5. 接口设计

### 5.1 输入接口
- **中间代码（四元式）**：来自中间代码生成器的输出

### 5.2 输出接口
- **目标代码**：作为JIT执行器的输入

### 5.3 错误处理接口
- **生成错误**：当遇到无法生成目标代码的情况时，报告错误

## 6. 实现细节

### 6.1 代码生成器架构

CScriptEngine 的目标代码生成器采用模块化设计，包含以下组件：

- **ICodeGenerator**：代码生成器接口，定义了所有代码生成器必须实现的方法
- **BaseCodeGenerator**：基础代码生成器，提供通用功能
- **X64Code**：x86-64 代码生成器实现
- **RV64Code**：RISC-V 64 代码生成器实现
- **CodeGeneratorFactory**：代码生成器工厂，根据目标平台创建相应的代码生成器

### 6.2 x86-64 代码生成

x86-64 代码生成的实现文件包括：
- **X64Code.cpp**：核心代码生成逻辑
- **X64CodeControl.cpp**：控制流指令生成
- **X64CodeMemory.cpp**：内存访问指令生成
- **X64CodeOps.cpp**：算术和逻辑操作指令生成
- **X64CodeUtils.cpp**：工具函数

主要功能：
- **指令集**：x86-64 指令集
- **寄存器**：通用寄存器（rax, rbx, rcx, rdx等）、浮点寄存器（xmm0, xmm1等）
- **调用约定**：x86-64 调用约定
- **内存访问**：RIP-relative 寻址
- **指令优化**：死代码消除、常量折叠等

### 6.3 RISC-V 64 代码生成

RISC-V 64 代码生成的实现文件包括：
- **RV64Code.cpp**：核心代码生成逻辑
- **RV64MemoryManager.cpp**：RISC-V 内存管理

主要功能：
- **指令集**：RISC-V 64 指令集
- **寄存器**：通用寄存器（x0-x31）、浮点寄存器（f0-f31）
- **调用约定**：RISC-V 调用约定
- **内存访问**：基址+偏移量寻址
- **栈帧布局**：符合 RISC-V ABI 规范

### 6.4 内存管理

内存管理组件负责分配和管理代码和数据的内存：
- **MemoryManager**：内存管理基类
- **WinMemoryManager**：Windows 平台内存管理实现
- **RV64MemoryManager**：RISC-V 平台内存管理实现

### 6.5 集成到编译器

目标代码生成器与中间代码生成器和JIT执行器集成，接收中间代码，生成目标代码后传递给JIT执行器。集成流程如下：

1. **中间代码生成**：IRGenerator 生成四元式序列
2. **代码生成**：CodeGeneratorFactory 创建目标平台的代码生成器，生成机器码
3. **JIT执行**：内存管理分配可执行内存，执行生成的代码
