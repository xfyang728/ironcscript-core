# 1. 简介

## 1.1 项目概述

CScriptEngine是一个轻量级的C语言脚本引擎，能够解析和执行C语言风格的脚本代码。它采用经典的编译器前端-后端架构，支持词法分析、语法分析、语义分析、中间代码生成、目标代码生成和JIT执行，同时支持生成x86-64、x86-32和RISC-V 64位机器码。

## 1.2 项目背景

随着嵌入式系统和物联网设备的普及，对轻量级脚本引擎的需求日益增长。传统的脚本引擎如Lua、Python等虽然功能强大，但在资源受限的嵌入式设备上运行时会占用较多的内存和CPU资源。CScriptEngine旨在提供一个轻量级、高效的C语言脚本解决方案，特别适合在资源受限的环境中使用。

## 1.3 项目目标

- **轻量级**：占用资源少，适合在嵌入式设备上运行
- **高效**：执行速度快，接近原生C代码的性能
- **跨平台**：支持多种平台，包括x86-64、x86-32和RISC-V 64位
- **易于集成**：提供简单的API，便于与其他系统集成
- **可扩展**：模块化设计，便于扩展和定制

## 1.4 项目范围

CScriptEngine专注于提供C语言的核心功能，包括：

- 基本数据类型（int、double、char、bool、float等）
- 控制流语句（if-else、while、for、do-while等）
- 函数定义和调用
- 数组和指针
- 基本的标准库函数

项目不包括：

- 复杂的C++特性
- 多线程支持
- 异常处理
- 操作系统特定的API
