# 2. 引用文档

## 2.1 相关标准

- **C99 标准**：ISO/IEC 9899:1999 - Programming languages — C
- **C11 标准**：ISO/IEC 9899:2011 - Programming languages — C
- **SDD 标准**：IEEE Std 1016-2009 - IEEE Standard for Information Technology—Systems Design — Software Design Descriptions

## 2.2 参考文档

- **《编译原理》**：Alfred V. Aho, Monica S. Lam, Ravi Sethi, Jeffrey D. Ullman. Compilers: Principles, Techniques, and Tools (2nd Edition). Addison-Wesley, 2006.
- **《现代编译原理》**：Andrew W. Appel, Jens Palsberg. Modern Compiler Implementation in C (2nd Edition). Cambridge University Press, 2004.
- **《C语言参考手册》**：Samuel P. Harbison, Guy L. Steele Jr. C: A Reference Manual (5th Edition). Prentice Hall, 2002.
- **《x86-64 汇编语言》**：Assembly Language for x86 Processors (7th Edition). Pearson, 2016.
- **《RISC-V 架构手册》**：The RISC-V Instruction Set Manual, Volume I: Unprivileged ISA. RISC-V Foundation, 2019.
